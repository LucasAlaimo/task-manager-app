# ✅ task-manager-app

A simple and practical task management web application built to organize daily tasks, track their status, and improve productivity.

---

## 🗺️ App Map

The project is organized into the main sections of a task management application.  
The content and features are still under development, and more functionality will be added over time.

```text
task-manager-app
├── Header
│   ├── App title
│   └── Short description
│
├── Task Form
│   ├── Task title input
│   ├── Task description input
│   ├── Priority selector
│   ├── Due date selector
│   └── Add task button
│
├── Task List
│   ├── Pending tasks
│   ├── Completed tasks
│   └── Empty state message
│
├── Task Item
│   ├── Task title
│   ├── Task description
│   ├── Priority label
│   ├── Due date
│   ├── Complete / uncomplete button
│   ├── Edit button
│   └── Delete button
│
├── Filters
│   ├── All tasks
│   ├── Pending tasks
│   ├── Completed tasks
│   └── Tasks by priority
│
├── Statistics
│   ├── Total tasks
│   ├── Pending tasks count
│   └── Completed tasks count
│
└── Local Storage
    ├── Save tasks
    ├── Load saved tasks
    └── Keep data after refreshing the page
```

## ✨ Features

- ✅ Add new tasks.
- 📝 Edit existing tasks.
- 🗑️ Delete tasks.
- ✔️ Mark tasks as completed.
- 🔎 Filter tasks by status.
- ⭐ Organize tasks by priority.
- 📅 Add due dates.
- 💾 Save tasks using local storage.
- 📱 Responsive design for desktop and mobile devices.
